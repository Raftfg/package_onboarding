# Package Onboarding Laravel (Fresh Build)

Package d'onboarding stateless sous le namespace `Raftfg`.

## Installation

1. Clonez ce dépôt.
2. Ajoutez le chemin vers ce dossier dans le `composer.json` de votre projet hôte :
```json
"repositories": [
    {
        "type": "path",
        "url": "./packages/raftfg/package_onboarding"
    }
]
```
3. Exécutez `composer require raftfg/package_onboarding`.

## Utilisation

Suivez les instructions du README dans le dossier `packages/raftfg/package_onboarding`.
